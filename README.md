# Relatório Financeiro

Este repositório contém relatórios e análises financeiras em formato Excel.

## Conteúdo

- **Relatórios Financeiro.xlsx** - Arquivo contendo dados e análises financeiras

## Como Usar

1. Baixe o arquivo `Relatórios Financeiro.xlsx` 
2. Abra com Microsoft Excel ou qualquer aplicação compatível
3. Analise os dados conforme necessário

## Estrutura do Arquivo

O arquivo Excel contém múltiplas abas com:
- Dados de receitas e despesas
- Análises e gráficos
- Planilhas de cálculos
- Relatórios consolidados
